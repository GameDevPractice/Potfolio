## 기술 문서
### Components
#### ActionComponent
Character들이 가지고 있는 무기에 따라 행하는 행동을 제어하는 Component입니다.  
가지고 있는 무기를 ActionType(Enum)으로 판단하며 Player는 이 ActionType을 통해 무기를 변경 할 수 있습니다.  
행하는 행동들은 DataAsset에 저장되어 있는 AnimMontage를 실행 합니다.
